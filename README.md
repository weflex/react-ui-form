# react-ui-form

The UI Component that providing basic form controls which is compatible with React.js

## Installation

```js
npm install react-ui-form
```

## Components

- Basis
  - `UIForm`
  - `UIRow`
- Buttons
  - `UIButton`
- Inputs
  - `UITextInput`
  - `UIDateInput`
  - `UITimeInput`
- Pickers
  - `UIOptionPicker`
  - `UIColorPicker`
